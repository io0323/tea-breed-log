# TeaBreed Log

茶の品種改良・育種試験記録システム

A tea breeding and variety testing record management system built with React + TypeScript + Vite.

## Features

- 品種情報の管理 (Variety information management)
- 育種データの記録と可視化 (Breeding data recording and visualization)
- 発芽率、生育スコア、耐病性の追跡 (Track germination rate, growth score, disease resistance)

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview

# Run linter
npm run lint
```

## Tech Stack

- React 19.2
- TypeScript 5.9
- Vite 7.2
- Tailwind CSS 4.1
- ECharts for data visualization
